XHTML & CSS

XHTML and CSS are code standards that make websites work. You can view the code of any webiste you visit.

Editors

Websites are basically text files. All you need is a simple text editor to make a website. You can use a Intergrated Development Environment.

It is unnessary to use it though.

Tags

Language of XHTML revovles around tags. First tag you create declares the doctype, or the type of document it will be.

Scope

Most tags have opening and closing tags. Everything in a web object is contained within the opening and closing tags.

Everthing else belongs o another object.

Head

It includes the information for the browser.

Body

This includes the actual thing you see on the site. <p> creates paragraphs since html doesn't trandlate spaces, tabs, and returns.

<br/> this is called line break. basically means it sperates paragraphs.

<hr/> creates a line that acts as a seperator.

<Strong> makes the word within bold.

<em> italicizes the word within.

Comments

You can add comments like all the other programs. You can also add links using <a>. That allows us to go to links on a page.

Header

You can add headers with <h(and the numbers 1-6)>. The smaller the number, the larger it is.
